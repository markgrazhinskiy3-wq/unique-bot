# Telegram Bot — Уникализация креативов для Facebook

Бот принимает фото и видео, применяет серию невидимых глазу преобразований и возвращает файл с другим хэшем (MD5/SHA), чтобы алгоритмы Facebook воспринимали его как новый уникальный креатив.

## Что делает бот

### Фото (JPEG, PNG, WEBP)
1. Уничтожение метаданных (EXIF, XMP, IPTC)
2. Сдвиг пиксельной сетки (смещение + обрезка + ресайз ±1-2px)
3. Субпиксельный шум (±1.5 на канал, невидим глазу)
4. Микросдвиг цветовых каналов (±1 на канал)
5. JPEG ghost (промежуточное сохранение с вариацией качества)
6. Микровращение (±0.3°)
7. Микрокорректировка резкости (97–103%)
8. Яркость/контраст (±0.3%)
9. Финальный ресайз ±1px

### Видео (MP4, MOV, AVI, MKV)
1. Удаление метаданных (`-map_metadata -1`)
2. Обрезка краёв (2–4px с каждой стороны несимметрично)
3. Ресайз обратно к оригинальному разрешению ±2px
4. Шум на видеопоток
5. Яркость/контраст/насыщенность ±0.5%
6. Вариация оттенка
7. Микровариация FPS (±0.1)
8. Темп аудио ±0.1%

## Запуск

### В Replit
Бот запускается автоматически через workflow "Telegram Bot".
Токен берётся из переменной окружения `TELEGRAM_BOT_TOKEN`.

### Docker
```bash
docker build -t unique-bot .
docker run -e TELEGRAM_BOT_TOKEN=your_token unique-bot
```

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather |

## Поддерживаемые форматы

- **Фото**: JPG, JPEG, PNG, WEBP
- **Видео**: MP4, MOV, AVI, MKV, WEBM

Рекомендуется отправлять файлы как документ (не сжатыми) для максимального качества.
